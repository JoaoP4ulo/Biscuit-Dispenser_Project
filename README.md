A project made using STM32 to automate the dosing and packaging process of a local company.

Project features:

- STM32F103C6
  -- 72MHz internal clock
  -- DMA-ADC input read
  -- GPIO-EXT input read
  --UART connection
  -- 60Hz PWM control

- Nextion HMI NX4024T032

- Custom PCB
